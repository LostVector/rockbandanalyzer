Rock Band Analyzer is hosted as a Google Code project.

You can obtain a read only copy of the current source svn checkout http://rockbandanalyzer.googlecode.com/svn/trunk/ rockbandanalyzer-read-only

The main library for Rock Band Analyzer and the command line executable wrapping the library are both written in Java using the 1.6 JDK.

The web frontend to the library was originally developed with Java Server Faces 1.1 as the UI framework. I discarded this web framework after discovering Apache Wicket. In addition, other components of the web frontend rely on the Google App Engine for Java SDK, since this application is hosted for free on Google App Engine.

Rock Band Analyzer also makes use of the JTidy and OpenCSV Java libraries. The project was developed with IntelliJ IDEA 8.1.
